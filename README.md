# Jatre-Namma Pride 🎉

## Project Title
**Android App Development using GenAI - Jatre-Namma Pride (State Pride)**

---

## 📖 Overview

Jatre-Namma is a digital guide for village fairs ("Jatre"), designed to help visitors and local communities stay informed, safe, and connected during large cultural gatherings.

The app provides real-time event schedules, lost & found support, parking guidance, safety information, and cultural stories related to the festival.

This project combines modern Android development with traditional rural culture to improve organization, public safety, and community engagement.

---

## 🚀 Features

### 📅 Live Event Schedule
- View ongoing and upcoming Jatre events
- Highlights the currently active event
- Real-time schedule updates using Firebase

### 🔍 Lost & Found
- Post details/photos of lost items or missing persons
- Mark reports as resolved
- Community-powered digital help desk

### 🅿️ Parking & Safety
- Locate parking zones
- Find first-aid and emergency help centers
- Improve crowd management during festivals

### 🗺️ Jatre Map
- Custom map with temporary festival stall markers
- Easy navigation inside the fair area

### 📚 Cultural Stories
- Learn about the history and legends behind the Jatre
- Promote cultural awareness among younger generations

# ⚙️ Installation & Setup

## Prerequisites

Before running the project, make sure you have:

- Android Studio installed
- Android SDK configured
- Kotlin support enabled
- Firebase project setup
- Internet connection

---

## 📥 Clone the Repository

```bash
git clone https://github.com/Deepthi123756/App-Devolopment-using-using-gen-AI
```

---

## 📂 Open the Project

1. Open Android Studio
2. Click **Open**
3. Select the cloned project folder

---

## 🔥 Firebase Setup

1. Create a Firebase project
2. Add an Android app to Firebase
3. Download the `google-services.json` file
4. Place it inside:

```text
app/google-services.json
```

---

## ▶️ Run the Project

1. Connect an Android device or start an emulator
2. Click the **Run ▶️** button in Android Studio
3. The app will build and launch automatically
